🎬 Movie Recommendation System

This project is a content-based Movie Recommendation System built using Python and Machine Learning techniques. It recommends movies based on similarity between movie features.

🚀 Features:
- Netflix-style user interface using Streamlit
- Movie recommendations using cosine similarity
- Real-time movie posters using TMDB API
- Movie ratings and overview
- YouTube trailer integration

🛠 Technologies Used:
- Python
- Pandas & NumPy
- Scikit-learn
- Streamlit
- TMDB API

📌 How it works:
The system uses content-based filtering by analyzing movie tags and computing similarity using cosine similarity. Based on user selection, it recommends similar movies.
